# 🚀 AI-Powered Data Pipeline & Analytics System

## 📌 Overview
Built an end-to-end data pipeline that ingests, processes, stores, and visualizes customer data with an interactive dashboard and automated scheduling.

## ⚙️ Tech Stack
- Python (Pandas, SQLAlchemy)
- PostgreSQL
- Streamlit
- Linux Cron / Task Scheduler

## 🔄 Features
- Automated ETL pipeline
- Data cleaning & transformation
- SQL-based analytics
- Interactive dashboard with filters
- Rule-based natural language query system
- Scheduled execution with logging
